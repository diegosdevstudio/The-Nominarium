# 🌌 The Nominarium
### *Elite Brand Identity & Naming Engine*

**The Nominarium** is a professional-grade AI tool designed for creative directors, brand strategists, and founders. Part of the **DiegosDevStudio** suite, it leverages the cutting-edge **Gemini 2.0 Flash** model to transform abstract brand visions into concrete identities, complete with curated names, brand philosophies, and visual directions.

---

## ✨ Key Features
* **Gemini 2.0 Integration:** Powered by the latest ultra-fast multimodal AI for superior linguistic creativity.
* **Luxury Essence Tags:** Fine-tune brand results using 10 specialized modifiers: *Heritage, Avant-Garde, Minimalist, Sustainable, Disruptive, Kinetic, Ethereal, Brutalist, Neo-Classic, and Bespoke.*
* **Brand Soul Cards:** Every generated name includes a "Brand Soul" (philosophy) and a "Visual Palette" (HEX codes and textures) for immediate creative direction.
* **Industrial Stability:** Built-in **Auto-Retry Logic** (handling 429 errors) and **JSON Sanitization** for a seamless, crash-free experience.
* **Signature UI:** A high-end "Midnight & Cyan" glassmorphism interface designed for professional environments.

---

## 🛠️ Technical Architecture
* **Core Model:** `gemini-2.0-flash`
* **Design System:** DiegosDevStudio "Black Label" (Midnight Navy #0B0F19 | Electric Cyan #22D3EE)
* **Security:** Client-side API key management via `localStorage`.
* **Performance:** Zero-dependency, single-file HTML/CSS/JS architecture for maximum speed and portability.

---

## 🚀 Quick Start
1.  Clone the repository.
2.  Open `index.html` in any modern web browser.
3.  Enter your **Google Gemini API Key** in the settings (⚙️).
4.  Input your brand vision, select your essence tags, and generate your identity.
